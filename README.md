# Typescript Playground

## Dependências
```bash
npm i -g typescript
npm i -g ts-node
```

OBS: É possível rodar arquivos `.ts` "nativamente", tendo as seguintes extensões:

1 - Code Runner
2 - JavaScript and TypeScript Nightly
